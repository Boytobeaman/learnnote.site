### Check Python Installation
```
python3 --version
```

### Install pip
```
Debian/Ubuntu:

sudo apt update  
sudo apt install python3-pip  
```

### For macOS
```
brew install python

// This will install the latest version of Python along with pip.
```


### Upgrade pip
```
python3 -m pip install --upgrade pip
```
