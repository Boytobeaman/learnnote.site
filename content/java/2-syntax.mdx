### Java 语法

在Java中运行的每一行代码都必须在内class。在我们的示例中，我们将类命名为MyClass。类应始终以大写首字母开头。
Java区分大小写：“ MyClass”和“ myclass”具有不同的含义。
Java文件的名称必须与类名称匹配。保存文件时，请使用类名进行保存，并在文件名的末尾添加“ .java”。



### 主要方法
