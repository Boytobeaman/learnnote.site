### Java快速入门

在Java中，每个应用程序都以一个类名开头，并且该类必须与文件名匹配。
我们创建第一个Java文件MyClass.java


MyClass.java
```
public class MyClass {
  public static void main(String[] args) {
    System.out.println("Hello World");
  }
}
```

### 编译代码
在控制台输入
```
javac MyClass.java
```

### 运行代码
```
java MyClass
```
